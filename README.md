# 📦 ระบบคลังสินค้าและติดตามสินค้าเฟอร์นิเจอร์เรียลไทม์ (Montipa & Motta Warehouse)

ระบบบริหารจัดการคลังสินค้าเฟอร์นิเจอร์และติดตามสินค้าแบบแยกคลังแบรนด์ **Montipa** (สีเขียวมิ้นต์ 🟢) และ **Motta** (สีชมพู 💗) ในรูปแบบ Single Page Application (SPA) ที่สวยงามด้วยสไตล์ **Glassmorphism** รองรับการทำงานแบบสมบูรณ์บนสมาร์ตโฟน (Mobile-Responsive 📱) พร้อมกลไกจัดเก็บข้อมูลออฟไลน์และความสามารถในการเชื่อมโยงฐานข้อมูลซิงค์คลาวด์ผ่าน Google Sheets / Google Drive

---

## 🌟 ฟีเจอร์หลักของระบบ (Core Features)

### 1. 🔍 ระบบตรวจสอบประวัติด่วน (Welcome Quick Scanner & Tracker)
* **การตรวจค้นข้ามแบรนด์ (Cross-Brand Search):** ค้นหาสินค้าจากเลขรหัสหรือการสแกนบาร์โค้ดได้ตั้งแต่หน้าแรกโดยไม่ต้องทำการเข้าสู่ระบบแบรนด์ก่อน
* **กลไกการสแกน Multi-device:** รองรับการเปิดกล้องโทรศัพท์มือถือสแกนโดยตรง (ผ่าน HTTPS) และมีแถบค้นหาแบบแมนนวล / อัปโหลดไฟล์รูปถ่ายบาร์โค้ดเพื่อถอดรหัสเป็นทางเลือกสำรอง
* **แบนเนอร์แสดงพิกัดแบบเร่งด่วน:** แสดงที่อยู่ปัจจุบันของชิ้นสินค้า เช่น *"ว่าง (อยู่ในคลังหลัก HQ)"* หรือ *"จัดแสดงอยู่ที่สาขา: Big C รัชดา ชั้น 1"*
* **ไทม์ไลน์ Stepper ประวัติการเดินทาง:** แสดงประวัติการทำธุรกรรมขนส่งและการเคลื่อนไหวทั้งหมดที่เกิดขึ้นกับสินค้าชิ้นนั้นจากอดีตจนถึงปัจจุบัน
* **ทางลัดเข้าแดชบอร์ดแบรนด์ (Quick Dashboard Access):** ปุ่มลัดพาผู้ใช้ออโต้ล็อกอินเข้าแบรนด์ สลับหน้า และเลื่อนจอไปหาตำแหน่งแถวสินค้าบนแดชบอร์ดพร้อมแสดงเอฟเฟกต์กะพริบ (Pulse Highlights) ทันที

### 2. 📊 แดชบอร์ดติดตามโครงการสาขา (Branch Project Tracking Dashboard)
* **Dropdown เลือกโครงการรองรับ 80+ สาขา:** คัดกรองรายชื่อสาขาที่ดึงข้อมูลขึ้นมาแสดงผลโดยตรงตามแบรนด์
* **แผงแผนงานโครงการ (Project Info Card):** แสดงประเภทงาน (Event, NEW Shop, Renovate), ขนาดพื้นที่, วันขนย้ายเข้า-ออก, ชื่ออินทีเรียผู้รับผิดชอบ และผู้ดูแลพื้นที่
* **คำสั่งเคลื่อนย้ายสินค้า (Interactive Logistics):**
  * ปุ่ม `📦 คลัง HQ` สำหรับย้ายของกลับศูนย์หลัก ล้างการจอง และปรับสถานะสินค้าเป็น `ว่าง` โดยอัตโนมัติ
  * ปุ่ม `🚚 ย้ายสาขา` สำหรับโยกย้ายเฟอร์นิเจอร์ระหว่างสาขาแบบเรียลไทม์
* **ประวัติโลจิสติกส์รายสาขา:** รายการประวัติการย้ายเข้า-ออกของอุปกรณ์ภายในสาขานั้นๆ

### 3. ⚙️ ระบบหลังบ้านและคำสั่งพิมพ์สติกเกอร์บาร์โค้ด (Single Sticker Printer Engine)
* **การแก้ไขรายละเอียดสินค้า:** หน้าต่าง Edit Modal พร้อมการเปลี่ยนเฉดสีแบบ Live Preview และสร้างเส้นบาร์โค้ดความละเอียดสูงผ่าน SVG
* **ระบบจัดทำสติกเกอร์เฉพาะตัว (Thermal Sticker Label):** จัดฟอร์แมตกระดาษความร้อนสำหรับเครื่องพิมพ์บาร์โค้ดโดยเฉพาะ ตัดส่วนเกินของเบราว์เซอร์ออก แสดงโลโก้, รหัสสินค้า, ขนาด กว้างxลึกxสูง, จุดสติกเกอร์สีจริง และบาร์โค้ดคมชัดสูง

### 4. ☁️ กลไกซิงค์คลาวด์ Dual-Mode (Google Sheets Integration)
* **No-Break Sync Policy:** ทำงานออฟไลน์ด้วย LocalStorage เวอร์ชัน 3.0 เสมอ และทำการเชื่อมต่อระบบซิงค์ออนไลน์ทันทีเมื่อผู้ใช้กำหนดค่า Apps Script Web App URL
* **แผ่นงานแยกอิสระ:** รองรับการสร้างชีตเก็บสินค้าและข้อมูลโครงการสาขาทั้ง `montipa_branches`, `montipa_furniture`, `motta_branches`, และ `motta_furniture` อัตโนมัติ

## 🛠️ รายละเอียดสแต็กเทคโนโลยี (Technical Stack)

1. **โครงสร้าง (Structure):** HTML5 (Semantic Elements, SPA Design)
2. **สไตล์และความสวยงาม (Styling):** Vanilla CSS (HSL Dynamic Theming, Glassmorphic Glass Panels, Responsive Flex/Grid Layouts, Pulse Animations)
3. **ตรรกะระบบ (Logic):** Vanilla JavaScript ES6 (LocalStorage Engine, SVG Barcode Generator, camera reader integration via `html5-qrcode`)
4. **ฐานข้อมูลและการซิงค์คลาวด์:** LocalStorage (คีย์ `kook_furniture_db_v3`) ร่วมกับ **Google Apps Script API**

## ⚙️ วิธีการเชื่อมโยงระบบเข้ากับ Google Sheets
1. เปิดบัญชี **Google Drive** และสร้าง **Google Sheets** ใหม่
2. ไปที่ **ส่วนขยาย (Extensions) > Apps Script**
3. คัดลอกโค้ดสำเร็จรูปที่อยู่ในหน้า **"ตั้งค่าระบบคลาวด์ชีต"** ของเว็บแอปนี้ไปวางและบันทึก
4. กด **Deploy (ทำให้ใช้งานได้) > New deployment (การทำให้ใช้งานได้ใหม่)**
   * เลือกประเภทเป็น **Web app (เว็บแอป)**
   * **Execute as:** Me (ตัวฉันเอง)
   * **Who has access:** Anyone (ทุกคน)
5. คัดลอกลิงก์ **Web App URL** ที่ได้มาวางลงในช่องบันทึกของตัวแอปพลิเคชันเพื่อเปิดใช้งานระบบออนไลน์!
